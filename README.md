# GlowCart 🌸

GlowCart is a full-stack MERN (MongoDB, Express, React, Node.js) e-commerce platform designed for skincare enthusiasts. It features product filtering, a wishlist, a shopping cart, and an admin dashboard to manage inventory.

## 🚀 Features
- **User Authentication:** Secure login and signup.
- **Product Filtering:** Filter by Category, Skin Type, and Concern.
- **Shopping Cart & Wishlist:** Manage your favorite products and orders.
- **Admin Dashboard:** Manage products and orders with ease.
- **Responsive Design:** Built with Tailwind CSS for a seamless mobile and desktop experience.

## 🛠️ Tech Stack
- **Frontend:** React, Tailwind CSS, Vite
- **Backend:** Node.js, Express
- **Database:** MongoDB
- **State Management:** Redux Toolkit
